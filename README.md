## Welcome to NoNeck
##
## Here are some basic Functions
### list - Lists all the current task in  your file
### delete (insert number) - delete the task inside the list depending on what number you input
### mark (insert number) - Mark the task you want as done depending on which number in the list
### unmark (insert number) - Unmark the task you want as done depending on which number in the list
### find (insert keyword) - It finds and list all the task whose description contains your keyword
### todo - Adds a Todo inside your list
### deadline (insert by date) - Add a deadline to your list as well as its date
### events (insert from date) (to date) - Add a event to yopur list with its from date and the to date
### bye - exits the application
